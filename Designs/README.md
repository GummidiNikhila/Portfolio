# Portfolio Website - Clean Code Version

A modern, responsive portfolio website for Nikhila G - Web Developer & Designer.

## 🚀 Features

- **Modern Design**: Clean, professional layout with gradient backgrounds and smooth animations
- **Responsive**: Fully responsive design that works on all devices
- **Interactive**: Smooth scrolling navigation, hover effects, and dynamic content
- **Performance Optimized**: Clean, organized code structure for better performance
- **Accessible**: Proper semantic HTML and ARIA labels for better accessibility

## 🛠️ Code Organization

### HTML Structure
- Semantic HTML5 elements (`<header>`, `<main>`, `<section>`, `<footer>`)
- Proper meta tags for SEO and social sharing
- Accessibility improvements with ARIA labels
- Clean navigation with proper linking

### CSS Architecture
- **Reset and Base Styles**: Consistent starting point
- **Header and Navigation**: Fixed header with smooth animations
- **Main Layout**: Flexible section layouts
- **Home Section**: Hero area with profile and introduction
- **About & Skills**: Cards with skill badges and highlights
- **Experience**: Timeline-style experience display
- **Projects**: Grid layout with hover effects
- **Contact**: Clean contact information cards
- **Footer**: Simple footer with social links
- **Animations**: Smooth transitions and hover effects
- **Responsive Design**: Mobile-first approach with media queries
- **Utility Classes**: Reusable spacing and alignment classes

### JavaScript Features
- **Typed.js Integration**: Animated typing effect for role descriptions
- **Smooth Navigation**: Section scrolling with active states
- **Section Headlines**: Dynamic section title updates
- **Scroll Animations**: Elements fade in as they come into view
- **Modular Functions**: Well-organized, reusable code
- **Event Handling**: Efficient event listeners

## 🎨 Design Features

### Color Scheme
- Primary: `#0ef` (Cyan)
- Secondary: `#08aeea` (Light Blue)
- Background: Dark gradient (`#232526` to `#0f2027`)
- Text: `#ededed` (Light Gray)
- Cards: `rgba(8, 28, 41, 0.8)` (Dark Blue with transparency)

### Animations
- Slide animations for header elements
- Fade-in effects for content sections
- Hover effects for interactive elements
- Smooth scrolling between sections
- Pulse effects for buttons

## 📱 Responsive Breakpoints

- **Desktop**: 1200px and above
- **Tablet**: 768px to 1199px
- **Mobile**: 480px to 767px
- **Small Mobile**: Below 480px

## 🚀 Getting Started

## 🔧 Customization

- Update contact details in the contact section
- Replace profile image in `images/assetss/`
- Modify project information in the projects section

### Styling
- Colors can be changed by updating CSS custom properties
- Animations can be disabled by removing animation classes
- Layout can be modified by adjusting grid and flexbox properties

### Content
- Add or remove sections as needed
- Update skill badges and experience items
- Modify project showcases
